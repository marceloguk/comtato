A J A X

A tecnologia AJAX

Permite atualização de regiões do documento de forma assíncrona. Ajax é a nomenclatura resultante das tecnologias Javascript + XML.

Os métodos 'responseXML' e 'responseText' recuperam informações diretamente de arquivos com extensões .xml e .txt respectivamente.

A verificação é feita por intermédio dos objetos:

  1. XMLHttpRequest() = IE7+,Firefox,Chrome,Opera,Safari
  2. ActiveXObject() = IE6,IE5

O método onreadystatechange possui a propriedade 'readyState'

  1. 0 = requisição não inicializada
  2. 1 = conexão com server estabelecida
  3. 2 = requisicão recebida
  4. 3 = requisição processada
  5. 4 = requisição finalizada e lida

O método onreadystatechange possui a propriedade 'status'

  1. 404 = Page não encontrada
  2. 200 = "OK"
