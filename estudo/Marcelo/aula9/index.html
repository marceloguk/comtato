Manifesto

  1. Crie um arquivo com extensão .manifest
  2. Associe o arquivo manifesto ao html < html manifest="arquivo.manifest" >
  3. Conteudo do arquivo:
    			CACHE MANIFEST
			# 2014-01-29

			#Arquivos a serem cacheados
			CACHE:
			/Roque/imagens/engrenagem.png
			/Roque/imagens/star.png
			/Roque/css/estilo.css
			/Roque/docs/offline-teste.html

			#Arquivos que nunca devem ficar em cache
			#E por sua vez não ficarão disponíveis offline

			NETWORK:
			*
			../css/Even Bader Mofo.ttf
  4. Para testar abra o arquivo html em localhost ou num servidor WEB. Depois desconecte o serviço ou a internet

C A N V A S

CANVAS 2d

Adicione imagens, crie desenhos, textos e efeitos de sombra. O tag canvas possibilita aceleração gráfica as imagens e videos contidos dentro dele.

Criando Círculos e Retângulos

O canvas permite a criação de objetos como círculos e retângulos. Seu navegador não tem suporte a CANVAS

Animação em CANVAS

Para animar o canvas utliza-se o método setInterval que repete a função em milisegundos. É necessário limpar o retangulo que é criado dinamicamente para não gerar peso de processo (clearRect)

Efeitos de composição (Mascaras)

Lighter,Darker,Top,Xor,Source-over entre outros. Criam efeitos de composição entre dois objetos.
